# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing us at:
**security@flyinfinitydao.io**

We will acknowledge your report within 24 hours and work to resolve the issue promptly.
