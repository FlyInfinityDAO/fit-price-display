# Fly Infinity Smart Contracts

This directory contains the smart contracts for the Fly Infinity platform, deployed on the BNB Smart Chain (BSC).

## 📋 Contracts Overview

| Contract | Address | Description |
|----------|---------|-------------|
| **Fly_Infinity_Token.sol** | `0xBebe4ea7aA4037CeF5D0bc25a10f9dcA16120Dfa` | Main FIT token contract with bonding curve pricing |
| **Fly_Infinity_Network.sol** | `0x7f1cB4fce7cB3b6C612c4A7Dd123D3865640dC1B` | Network contract for ownership and rewards |
| **Fly_Infinity_Gift.sol** | `0x0fb7b50c57d7E7627a7C4584A5BbbbAb6ee17CE8` | Gift and lottery contract |
| **Fly_Infinity_DAO.sol** | `0xB56c380287b7Fdc9940501f13Fc185A787a6F5D9` | DAO governance contract |
